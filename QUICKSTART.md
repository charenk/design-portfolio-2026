# Quick Start Guide

Get your portfolio website running in 3 minutes! 🚀

## 🏃 Fast Track

```bash
# 1. Navigate to project
cd "/Users/charen/Documents/5.Personal stuff/portfolio-test"

# 2. Install dependencies (if not already done)
npm install

# 3. Build CSS
npm run build

# 4. Start server
npm run serve
```

Open http://localhost:8000 in your browser. Done! 🎉

## 📱 View Your Site

The server is already running on:
**http://localhost:8000**

Test on different devices:
- Desktop: Open in browser
- Mobile: Visit from phone on same network using your computer's IP

## 🎨 Making Changes

### Update Content
1. Edit `index.html` (text, links, images)
2. Save file
3. Refresh browser

### Update Styles
1. Edit `src/input.css` or `tailwind.config.js`
2. Run: `npm run build`
3. Refresh browser

### Update Interactivity
1. Edit `src/main.js`
2. Save file
3. Refresh browser

## 🚀 Deploy to Web

### Easiest: Netlify Drop
1. Run `npm run build`
2. Go to https://app.netlify.com/drop
3. Drag entire folder
4. Get instant live URL!

### Best: Git + Netlify/Vercel
1. Push to GitHub
2. Connect to Netlify/Vercel
3. Auto-deploy on every push

## 📁 Key Files

- `index.html` - Main page
- `src/input.css` - Styles source
- `src/main.js` - JavaScript
- `tailwind.config.js` - Design tokens
- `assets/` - Images
